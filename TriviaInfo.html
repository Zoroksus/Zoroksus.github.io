My Information Website for Trivia Game

- Value Proposition

For people who are interested in the history of the Origami art, this trivia game is a quiz that can be interacted with to help develop an understanding of how the art came to be what it is today.

- Features -

- Perhaps music

- Idle interactions (things moving on the screen, when the mouse is over a button, it pulses, etc.)

- Questions

- A unique answer layout (how the answers are presented)

- A poll screen at the end

- Multiple “screens” and game modes

- Game modes: “speed quest” (answer as many as possible in “x” seconds), etc.
